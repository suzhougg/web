简要描述

MVC（Model-View-Controller） 是 PHP 开发中常用的架构模式，大部分项目构建都是基于这种模式。

  • 模型 Model – 连接和操作数据库数据。
  • 控制器 Controller - 负责响应用户请求、准备数据，以及决定如何展示数据。
  • 视图 View – 负责渲染数据，通过 HTML 方式呈现给用户。
